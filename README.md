
# Currency Converter

I developed an app that getting json data from "https://jsonplaceholder.typicode.com/". 
I used the MVVM design pattern. I seperated the UI, Data and BL.
## Used

**MVVM Design Pattern** ✅

**Json Parsing** ✅

**Segue** ✅

**UITableViewCell** ✅
